1. To build the required files
    `make build`

2. To generate root and intermediate certificates
    `make ca-certsgen`

3. To generate client keys, certs, and config
   `make client-keygen`
   `make client-certsgen`
   `make client-config-gen`

4. To generate client keys, certs, and config (sudo command)
   `make server-keygen`
   `make server-certsgen`
   `make server-config-gen`
